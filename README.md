# opencode_project

Minimal README for the project.

Overview
- This repository contains code for OpenManus and workspace components.

Source control
- Default branches: `master` (production) and `develop` (active development).
- See `CONTRIBUTING.md` for branching and PR workflow.

Quick start
1. Create and activate a Python virtual environment.
2. Install dependencies: `pip install -r requirements.txt` (if present).
3. Run the main script: `python main.py` or `python run_bot.py` as needed.

Contributing
- Use feature branches, open PRs against `develop`, and request reviews.
